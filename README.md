# 豆包提示词助手 Chrome 插件

## 功能说明

该插件用于在访问豆包聊天页面（https://www.doubao.com/chat/）时，快速插入用户配置的提示词到聊天输入框中。

## 安装步骤

1. 克隆或下载本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/` 页面
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目的根目录
5. 插件安装成功后，会在浏览器右上角显示插件图标

## 使用方法

### 添加提示词
1. 点击浏览器右上角的插件图标
2. 点击「管理提示词」按钮，进入选项页面
3. 在选项页面中，输入提示词名称和内容
4. 点击「添加提示词」按钮，保存提示词

### 使用提示词
1. 访问豆包聊天页面（https://www.doubao.com/chat/）
2. 点击浏览器右上角的插件图标
3. 在弹出的提示词列表中，点击需要使用的提示词
4. 提示词会自动插入到豆包的聊天输入框中

## 插件结构

```
xhs_plugin_helper/
├── manifest.json        # 插件配置文件
├── content.js           # 内容脚本，用于注入页面功能
├── popup.html           # 插件弹出页面
├── popup.js             # 弹出页面逻辑
├── options.html         # 选项页面，用于管理提示词
├── options.js           # 选项页面逻辑
├── images/              # 插件图标目录
└── README.md            # 说明文档
```

## 注意事项

- 提示词存储在 Chrome 的同步存储中，可以在不同设备间同步
- 每个提示词的内容长度建议不要过长，以免影响插入效果
- 插件仅在访问豆包聊天页面时生效
